## Instruções de instalação e execução
### Pré-requisitos
- Node.js >= X
- Java >= X (se backend em Spring Boot)
- Banco de dados: PostgreSQL

### Passos
1. Clonar repositório: `git clone <URL>`
2. Backend:
   - Navegar para `backend/`
   - Configurar variáveis de ambiente (EXEMPLO: DB_URL, DB_USER, DB_PASS)
   - `./mvnw spring-boot:run`  ou `npm install && npm start` conforme aplicável
3. Frontend:
   - Navegar para `frontend/web/`
   - `npm install`
   - `npm start`
4. Database:
   - Executar `database/schema.sql` para criar tabelas
